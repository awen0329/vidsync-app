# Vidsync v0.0.1

First desktop release of Vidsync — peer-to-peer video file sync for
production teams.

## Highlights

- **Peer-to-peer sync** of project folders between devices with no size
  limit and no cloud upload, running at LAN speed when crew is on-site.
- **Per-project membership** — invite collaborators by email; each project
  has its own access control.
- **Media-aware file grid** with thumbnails and in-app previews, including
  transcoded playback for non-browser formats (MXF, BRAW, etc.).
- **Dropbox import/export** for moving footage in and out of a project.
- **Cross-platform desktop app** (Windows, macOS) with a system-tray
  background agent, auto-start, and a built-in updater hook.
- Device verification and account sign-in via the Vidsync cloud control
  plane.

## Notes

- Versions are injected at build time; see `build-release.ps1`
  (Windows) and `build-release-darwin.sh` (macOS).
- Vidsync's sync engine is built on Syncthing (MPL-2.0); see `LICENSE`
  and `AUTHORS`.
